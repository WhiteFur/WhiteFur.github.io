Integrate disqus into wintersmith

Written by whiteFur

剛架好Wintersmith後發現，咦？奇怪！怎麼沒有留言功能

原來留言功能是要另外接上去的，看了很多人Blog，大多數人都用disqus來管理留言。

disqus的安裝很簡單

  1. 到disqus註冊一個website

  2. 註冊完後，它會列出disqus支援的各家Blog service(ex:Blogger, Wordpress…)，選擇Universal Code，它會給你一段html code, 把它們放到每一頁文章使用的template裡即可。

我個人是用jade當做template。頁面的metadata如下

---
title: Integrate disqus into wintersmith
author: whiteFur
date: 2014-07-09 10:02
template: article.jade
---

所以我必需把disqus embedded code 放到 article.jade

jade也可以擺html，但是我喜歡整齊一點，所以把code都轉成jade的格式，放到disqus.jade，再include到article.jade。

disqus.jade (把disqus_shortname的值換成step1註冊時填的unique Disqus URL即可)

mixin disqus()
  div#disqus_thread

  script.
    /* * * CONFIGURATION VARIABLES: EDIT BEFORE PASTING INTO YOUR WEBPAGE * * */
    var disqus_shortname = 'whitefur'; // required: replace example with your forum shortname

    /* * * DON'T EDIT BELOW THIS LINE * * */
    (function() {
        var dsq = document.createElement('script'); dsq.type = 'text/javascript'; dsq.async = true;
        dsq.src = '//' + disqus_shortname + '.disqus.com/embed.js';
        (document.getElementsByTagName('head')[0] || document.getElementsByTagName('body')[0]).appendChild(dsq);
    })();

  noscript.
    Please enable JavaScript to view the <a href="http://disqus.com/?ref_noscript">comments powered by Disqus.</a></noscript>
    <a href="http://disqus.com" class="dsq-brlink">comments powered by <span class="logo-disqus">Disqus</span></a>

article.jade

block prepend footer
  include disqus

  div.nav
    a(href=contents.index.url) « Full blog
  mixin disqus()
